Principais Formatações

Negrito/Destaque

Nessa frase tem um termo em negrito de forma não semântica usando a tag <b>.

Já nesta frase temos um termo em destaque de forma semântica usando a tag <strong >. Já que esta usa seu sentido semântico para destacar, não apenas a forma. Quando estamos falando em sentido semântico, queremos usar o negrito como sentido de destaque.

Itálico / Ênfase

Nesta frase temos um termo em itálico de forma não semântica usando a tag <i>.

Já nessta frase temos um termo em ênfase usando a tag <em>. Já que esta usa seu sentido semântico para destacar, não somente a forma.Quando estamos falando em sentido semântico, queremos usar o itálico como sentido de ênfase.
